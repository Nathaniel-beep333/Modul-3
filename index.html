Mulai Praktikum

Ringkasan Materi

Bootstrap menyediakan grid responsif, komponen siap pakai (navbar, card, form) dan utilitas CSS yang mempercepat pembuatan tampilan web.

  • Grid system: row & col-*
  • Komponen: Navbar, Card, Button, Form
  • Utilities: spacing, text, display, alignment
Struktur HTML

Memahami tag, atribut, dan struktur dokumen HTML yang semantik untuk halaman web yang terorganisir.
